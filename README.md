# webSafe
webSAFE is a Secure and Free Encryption Decryption Online service in Javascript.

This uses AES 256 cipher to allow you to encrypt plain text using just a simple keyphrase. 

This project was built using:
<ul>
<li>CryptoJS</li>
<li>JS Packer</li>
<li>JS FileSaver</li>
